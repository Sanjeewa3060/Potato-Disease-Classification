Project Overview

About LeafGuard AI

LeafGuard AI is an advanced decision-support platform designed to help farmers, researchers, and agronomists identify crop diseases at early stages. Early diagnosis prevents crop damage, reduces pesticide usage, and secures harvests.

Dataset preprocessing with augmentation techniques to address class imbalance.
CNN model architecture compiled using TensorFlow and Keras backends.
FastAPI backend delivering lightning-fast, high-concurrency REST endpoints.
Clean, modular, accessible and highly responsive static frontend.

This project was built for educational and research purposes as a demonstrating prototype of deep learning in smart agriculture.
